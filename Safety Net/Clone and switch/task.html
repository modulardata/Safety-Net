Description

Let's start with a simple task. Clone the project repository from the remote and switch to the development branch.

You may corrupt the repository while working on this project. As a precaution, it may be helpful to understand the status of the repository and branches before starting the objectives. The commands git branch --all and git log --oneline can help you.
If you want to start over, you can clone the repository again.

  • Switch to the 0.2.x-dev branch (create a local copy from the remote).
