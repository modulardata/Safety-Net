Description

Since your work on the feature branch has finished, you can merge it into the development branch. After that, delete the source branch.

Objectives

Let's break this task into a few steps:

  • Since the development branch has changed, first, rebase feature/case branch with the0.2.x-dev branch to reflect the changes;
  • Switch to the 0.2.x-dev branch;
  • Merge feature/case branch into it;
  • Delete the feature/case branch.
